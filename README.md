# ServerStatusProbe
A probe for your server - 您的服务器探针

## 项目介绍
SSProbe 是一个探针 他可以对您的服务器进行一个多方面的管理

## 功能列表
> 1. 完整的数据检测功能 [CPU RAM 磁盘 网卡 等信息的实时监控] <br>
> 2. 在线使用服务器的 Shell <br>
> 3. 文件管理 [待定] <br>
